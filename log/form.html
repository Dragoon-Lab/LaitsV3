Help for the above functionalities

  • Time On Task & Solution Verification uses section and student name. Student name is for a particular user id that you want to check the solution time for particular user. If you want to check for whole section then select one of the two possible values from the Section Dropdown.
  • In Time-on-Task if you specify then we find the problems that have been worked out during that time by all the users possible. If you specify a particular user, problem and time then we work as per the problems.
  • Class Problems finds all the problems that student have worked in a particular time and gives the JNLP urls to test all the final solutions of the problems. You can specify time / sections as per your need. If no section is specified then all the problems are enlisted for the students in both the sections. It doesnt work to find the problems worked by a single user. If no time is specified then a default time of November 6th, 2013 3 pm to 4 30 pm is used.
  • Check Logs - specify a particular log table that you want to check. It will return all the logs from the table specified in that time. As this can be really heavy if one does not specify the time, so you can see only logs from last 7 days only, if you dont specify the time.
