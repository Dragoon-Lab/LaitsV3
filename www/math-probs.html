Function Description
sin(x) Sine of x (x is in radians)
cos(x) Cosine of x (x is in radians)
Tan(x) Tangent of x (x is in radians)
asin(x) Arc sine of x (x is in radians)
acos(x) Arc Cosine of x (x is in radians)
atan(x) Arc Tangent of x (x is in radians)
atan2(x, y) Arc Tangent of x/y. i.e. the angle between (0, 0) and (x, y) in radians.
sqrt(x) Square Root of x. Result is NaN (Not a Number) if x is negative
log(x) Natural Logarithm of x (following the javascript naming convention)
abs(x) Absolute Valueof (magnitude) of x
ceil(x) ceilingof x - the smallest integer that's >= x.
floor(x) Floor of x - the largest integer that's <=x
round(x) x, Rounded to nearest integer, using "gradeschool rounding"
exp(x) Exponential of x. ie e^x
random(n) Get a Random Number in the range [0, n). If n is zero, or not provided, it defaults to 1.
fac(n) n! Factorial of n: “n * (n-1) * (n-2) * … * 2 * 1″) or n!
min(a,b,…) Get the smallest (“minimum”) number in the list
max(a,b,…) Get the largest (“maximum”) number in the list
pyt(a, b) Pythogorean function, i.e. the c in “c2 = a2 + b2“
pow(x, y) Power Function Returns “x^y”. It’s just provided since it’s in the Math object from JavaScript
pulse(x,y) pulse of x , y is equal to 1 between times x and y inclusively.
pulsetrain(x,y,z) pulse train of x,y,z is equal to one starting at x for a length of y and then equal to zero for a length of z and then repeats.
